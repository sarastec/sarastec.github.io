General info

This is a website I made to showcase my projects and experience. It also serves as an extension to my resume for anyone who wants to read up on details of projects I participated in. My aim was to make a minimalist website with a clear layout so that accessing desired information is fast and easy.

Technologies

  • Nuxt
  • Tailwind

This is a small static website that I definitely didn't need to use Nuxt nor Vue for. The reason I did it was because of another project I was working on at the time - Quanta. Quanta is a team project that we decided to use Nuxt to create, but it was my first time using this technology. At the same time I needed to make this website, so I decided to make it in Nuxt as well, so I can learn and work more efficienty on Quanta later.

Tailwind CSS was used for most of this website's formatting. It was also my first time using it. The reason I learned it was because I saw it recommended as an easy to pick up, efficient tool that made the code a lot easier to read, especially for small websites like mine.

Setup

If you want to download the repository and run it on localhost, these are the steps:

  • Open the project directory
  • Run npm run dev
  • Open the corresponding localhost port in browser

Build and Github Pages deployment was made using this method.

Status

I plan on keeping this website up to date with new projects and updated information indefinitely.
